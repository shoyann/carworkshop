# 车机创意工坊 · MVP

## 你有什么

```
server.js          ← 服务器（大脑）
public/
  control.html     ← 手机/电脑上操作的页面
  tablet.html      ← 平板上全屏显示的页面
```

---

## 启动（让硬件专家帮你做一次）

```bash
# 1. 进入项目目录
cd carworkshop

# 2. 启动服务器
node server.js

# 3. 看到这行就成了：
#    Car Workshop Server running on http://localhost:3000
```

---

## 使用流程

**步骤 1：平板（放车里）**
- 打开浏览器，访问 `http://[树莓派IP]:3000/tablet.html`
- 全屏，放在仪表台上
- 页面会显示"等待推送"

**步骤 2：你的手机/电脑**
- 访问 `http://[树莓派IP]:3000/control.html`
- 选风格 → 选灯光颜色 → 点「生成开机动画」
- 有可灵 API Key 就填，没有会用演示视频
- 点「发送到车内」

**步骤 3：平板上自动播放！**

---

## 可灵 API Key 怎么拿

1. 去 https://klingai.com 注册
2. 后台找 API 管理，创建 Key
3. 粘贴到控制页面的输入框

每次生成大概等 60-90 秒（可灵生成视频的时间）

---

## 树莓派 IP 怎么查

```bash
hostname -I
```
